# BL.SC.P2CSE25027

This repository contains my solution for the AffordMed coding test.

I implemented a backend using Node.js and Express and a simple frontend using HTML, CSS, and JavaScript. The APIs handle basic operations like adding and retrieving data.

The project is structured in a simple way to keep things clear and easy to run.

## How to run

1. Install dependencies
   npm install

2. Start the server
   node server.js

3. Test APIs using Postman or browser
